# Stellar Burger

Учебный проект космической бургерной, разработанный в рамках курса Яндекс.Практикума по веб-разработке. Приложение позволяет собирать бургеры из ингредиентов, оформлять заказы и отслеживать их статус в реальном времени.

---

## Деплой

Проект доступен по адресу:
[https://burgerjoint.students.nomorepartiessite.ru](https://burgerjoint.students.nomorepartiessite.ru)

---

## Стек технологий

- **React** — библиотека для построения пользовательских интерфейсов
- **TypeScript** — статическая типизация JavaScript
- **Redux Toolkit** — управление состоянием приложения
- **React Router v7** — маршрутизация и навигация
- **WebSocket** — получение заказов в реальном времени
- **Vite** — инструмент сборки проекта
- **CSS Modules** — модульная стилизация компонентов
- **Nginx** — веб-сервер для раздачи статических файлов
- **Яндекс.Облако** — облачная платформа для хостинга

---

## Функциональность

- Авторизация и регистрация пользователей
- Редактирование профиля
- Обновление токенов
- Добавление ингредиентов в конструктор бургера
- Создание и оформление заказа
- Просмотр ленты заказов в реальном времени через WebSocket
- История заказов пользователя

## Запуск проекта локально

```bash
# Установка зависимостей
```
